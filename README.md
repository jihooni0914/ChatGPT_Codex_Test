# ChatGPT Codex Test

This repository contains a simple Python script to visualize university entrance grade cutoffs.

## Usage

First, install the required dependencies:

```bash
pip install pandas matplotlib
```

Run the visualization script specifying the year (2023 or 2024):

```bash
python visualize_grades.py --year 2024
```

The script reads data from `grades.csv` and displays a bar chart comparing different admission types.
